Xero Accounting System Automation

Written in Python on the Pycharm interpreter, this program automated invoice batching and payment integration using the Xero API

LinkedIn Job Search and Application robot

Efficient Frontier Calculator

Written in Python on the Jupyter Lab Google Colab application, this program uses Modern Portfolio Theory (Markowatz 1952) to analyse a portfolio of investments and calculate the Efficient Frontier for the investment portfolio. This program uses several Python libraries such as matplotlib, Numpy, Pandas and webscraped up to the second financial data to build the analysis at the push of a button

Investment Portfolio Backtesting Tool

Written in Python on the Jupyter Lab Google Colab application, this program expands on the Efficient Frontier calculator to calculate, over a selected period, what the results of the portfolio investment would have been had you followed the EF strategy applied to a portfolio of assets

Monte Carlo Simulation of an Investment Portfolio of Assets

Written in Python on the Jupyter Lab Google Colab application, this program expands on the Efficient Frontier calculator and determines the optimised best expected investment weights of all the assets in a portfolio that would yield the highest returns forcasted for one year. The forcast is achieved through the modeling of 10,000 Monte Carlo simulations and selecting the best perorming portfolio based on the highest Sharpe Ratio
